// Project Plan · College Submission

Employee Payroll
Management System

A full-stack web application to manage employee salaries, deductions, taxes, leaves, and auto-generate PDF payslips — with role-based access for HR, Manager, and Employee.

React + Bootstrap Java + Spring Boot PostgreSQL JWT Auth PDF Generation REST API Email Notify
// 01 — User Roles
Who Uses This System?
🛡️
Admin
Full access — manage employees, departments, salary structure, and system config
👔
HR Manager
Add/edit employees, approve leaves, process monthly payroll, send payslips
🧑‍💼
Employee
View own payslips, salary breakdown, leave balance, and apply for leaves
// 02 — Core Modules
System Features & Modules
👤
Employee Management
  • Add / edit / deactivate employees
  • Department & designation assignment
  • Employee ID & profile photo upload
  • Bank account details for salary credit
💰
Salary Management
  • Define salary structure (Basic, HRA, DA, TA)
  • Configure deductions (PF, ESI, Tax)
  • Increment & revision history
  • Net salary auto-calculation
📄
Payroll Processing
  • Monthly payroll run with one click
  • Attendance-based pay deduction
  • Overtime pay calculation
  • Payroll approval workflow
🧾
Payslip Generation
  • Auto-generate PDF payslips
  • Company logo & letterhead on slip
  • Email payslip to employee inbox
  • Download/print from portal
🏖️
Leave Management
  • Apply for Casual / Sick / Earned leave
  • Manager approval workflow
  • Leave balance tracker
  • Leave deduction in payroll
📊
Reports & Analytics
  • Monthly payroll summary report
  • Department-wise salary charts
  • Tax deduction report (Form 16)
  • Export reports as PDF / Excel
// 03 — Standout Features
What Makes It Impressive?
📧
Auto Email Payslips
Spring Boot + JavaMailSender sends PDF payslips to all employees every month automatically
🔐
JWT Role-Based Auth
Secure login with JWT tokens; each role (Admin / HR / Employee) sees only their own views
📊
Live Dashboard Charts
React Chart.js dashboard showing payroll costs, department breakdown, and monthly trends
🧮
Tax Auto-Calculator
Automatically computes income tax slabs (old/new regime) and shows annual tax liability

// 05 — REST API Endpoints
Spring Boot API Design
POST/api/auth/loginJWT login for all roles
GET/api/employeesList all employees (HR/Admin)
POST/api/employeesAdd new employee
PUT/api/employees/{id}Update employee details
GET/api/salary/{emp_id}Get salary structure
POST/api/payroll/runTrigger monthly payroll processing
GET/api/payslip/{emp_id}/{month}Download PDF payslip
POST/api/leaves/applyApply for leave
PUT/api/leaves/{id}/approveApprove or reject leave
GET/api/reports/payroll-summaryMonthly payroll report

// 07 — Development Timeline
4-Week Build Plan
WEEK 01
Project Setup + Auth + Employee Module
Set up Spring Boot project, PostgreSQL DB, JPA entities. Build JWT login API. Create Employee CRUD — add, edit, view, deactivate employees with department assignment.
WEEK 02
Salary Structure + Leave Management
Define salary components (Basic, HRA, DA, deductions). Auto net-salary calculation. Build leave application & approval flow. Integrate leave deductions into payroll logic.
WEEK 03
Payroll Processing + PDF Payslip + Email
Monthly payroll run API. Generate payslip PDFs with iText (company logo, salary breakdown). JavaMailSender to email payslips. Employee portal to download own payslip.
WEEK 04
Dashboard, Reports, UI Polish & Testing
React dashboard with Chart.js graphs. Monthly payroll summary report (PDF/Excel export). Complete UI with Bootstrap 5 responsive design. Postman API docs. Final testing & bug fixes.
